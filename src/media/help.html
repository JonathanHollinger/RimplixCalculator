Simple Solutions for Complex Numbers


Calculator numberpad

  1. ± - Changes the number from positive to negative and vice versa.
  2. C - Clears the display of the Calculator
  3. ← - Removed the previously entered key
  4. + - Adds values together
  5. R - Resets the calculator
  6. - - Subtracts values together
  7. Inv - To be determined
  8. ( - Creates a new operand
  9. ) - Finishes an operand
  10. ÷ - Divides values together
  11. i - Adds an imaginary number to the operand
  12. = - Performs the equation saved
  13. . - Creates a decimal number, allowing for more detailed calculations
  14. 1-9 - Adds the respective number to the operand
